- 刪除節點：
- 刪除節點時，需考慮三種情況：無子節點、只有一個子節點、有兩個子節點。
- 若只有一個子節點，則直接將該子節點取代被刪除的節點。
- 若有兩個子節點，則找到右子樹中的最小節點（或左子樹中的最大節點），將其值取代被刪除的節點，然後再刪除該最小（或最大）節點。
